## ✨ Возможности

- 🔍 **Автоматический поиск спектаклей** — парсинг актуальной афиши театра
- 💺 **Интеллектуальный выбор мест** — бронирование лучших доступных мест по заданным параметрам
- ⏱ **Планирование запусков** — встроенный scheduler для работы по расписанию
- 🖥 **Полная эмуляция браузера** — используется Selenium с WebDriver для надёжного взаимодействия
- 📊 **Логирование** — подробные логи всех этапов работы парсера
- 🔧 **Гибкая настройка** — конфигурация через переменные окружения или файл настроек

## 🚀 Как это работает

Парсер имитирует действия реального пользователя:
1. Открывает сайт театра через браузер (управляемый WebDriver)
2. Навигируется по разделам афиши
3. Выбирает заданный спектакль
4. Автоматически подбирает места согласно предпочтениям
5. Инициирует процесс бронирования

Встроенный планировщик позволяет запускать проверку доступности билетов в нужное время — например, сразу после открытия продаж.

## ⚙️ Установка и запуск

### Требования
- Python 3.8 или выше
- Microsoft Edge (или Chrome/Firefox — настраивается)

### Шаги по установке

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Artpardsur/theater_parser.git
cd theater_parser

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Запустите парсер
python main.py

# Для работы по расписанию используйте:
python scheduler.py
```

**Важно**: Убедитесь, что msedgedriver.exe или драйвер для вашего браузера находится в PATH или в корне проекта. Для других браузеров обновите настройки в коде.

## ⚙️ Конфигурация

Создайте файл .env или отредактируйте src/config.py:

```ini
# Настройки браузера
BROWSER_TYPE=edge          # edge / chrome / firefox
HEADLESS_MODE=false        # true — без графического интерфейса

# Настройки спектакля
TARGET_PLAY=Ревизор         # Название спектакля для поиска
PREFERRED_SECTIONS=Бельэтаж,Амфитеатр  # Предпочтительные секции
MAX_PRICE=1500              # Максимальная цена билета

# Расписание (для scheduler.py)
SCHEDULE_TIME=10:00         # Время запуска
CHECK_INTERVAL=30           # Интервал проверки в минутах
```

## ⏰ Планирование задач

scheduler.py позволяет запускать парсер автоматически по расписанию:

```bash
# Запуск планировщика
python scheduler.py
```

Планировщик будет выполнять проверку доступности билетов с заданной периодичностью. Это удобно, если вы ждёте появления билетов на определённую дату или хотите поймать отменённые брони.

### Запуск планировщика
python scheduler.py

Планировщик будет выполнять проверку доступности билетов с заданной периодичностью. Это удобно, если вы ждёте появления билетов на определённую дату или хотите поймать отменённые брони.

### Пример настройки планировщика

```python
# scheduler.py
import schedule
import time
from main import run_parser

# Запуск каждый день в 10:00
schedule.every().day.at("10:00").do(run_parser)

# Запуск каждые 30 минут в течение дня
schedule.every(30).minutes.do(run_parser)

while True:
    schedule.run_pending()
    time.sleep(1)
```

### 🐛 Известные проблемы и их решение

| Проблема | Решение |
|----------|---------|
| WebDriver не найден | Убедитесь, что драйвер установлен и добавлен в PATH |
| Сайт театра изменил структуру | Обновите селекторы в `src/parser.py` |
| Браузер не закрывается после завершения | Проверьте логи, возможно, произошла ошибка — добавьте `finally: driver.quit()` |
| Не удаётся найти спектакль | Проверьте точное название в настройках `TARGET_PLAY` |
